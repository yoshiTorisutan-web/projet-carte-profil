# Carte de Profil Animée

Une carte de profil moderne et interactive réalisée en **HTML, CSS (Tailwind)** et **JavaScript** avec une charte graphique verte. Le projet met en avant un design **minimaliste et moderne**, idéal pour un portfolio ou une page “À propos”.

---

## Fonctionnalités

- Carte interactive avec **hover animé** et **effet 3D léger**
- Photo de profil ronde avec bordure verte et animation
- Bio courte et pertinente
- Compétences affichées sous forme de tags
- Expérience rapide
- Boutons sociaux animés (LinkedIn, Portfolio, GitHub)
- Design moderne avec **glassmorphism**
- Favicon rond vert intégré

---

## Technologie utilisées

- [Tailwind CSS](https://tailwindcss.com/) pour le design
- HTML5 et CSS3
- JavaScript minimal pour l’interaction 3D
- Font Awesome pour les icônes
